## Video Lectures

- **Video 1: History and Evolution of LLMs** — Traces LLM evolution from ELIZA's rule-based system in the 1960s, through statistical methods in the 1990s, neural networks in the 2000s, and transformers in 2017, to today's more capable models.
- **Video 2: Types of LLMs** — LLM types by access, purpose, and architecture (GPT-4, BERT, LLaMA); self-supervised training; the move toward efficient, flexible, smarter models.
- **Video 3: Building Blocks of LLMs** — Tokens as the basic units LLMs use; how tokenization breaks down text and why different models handle the same text differently.
- **Video 4: Context Window** — How much information an LLM can process at once; comparison of GPT-3.5, GPT-4 and Claude; how context size affects performance and resource use.
- **Video 5: Demonstrating Code Generation with ChatGPT** — Five ways generative AI boosts software development: prototyping/debugging with ChatGPT, building websites with Claude, GitHub Copilot assistance, integrating OpenAI APIs, and automating web data analysis.
- **Video 6: Creating a Website Using Claude AI** — How Claude AI enables fast, low-code website creation from natural language prompts — generating, previewing, refining and publishing HTML/React sites.
- **Video 7: Code Generation and Modification Using Copilot** — How GitHub Copilot (with Visual Studio or Jupyter Notebooks) supports writing, editing and documenting code — generating Python (FizzBuzz), debugging logic errors, auto-generating documentation.
- **Video 8: Data Extraction Using the OpenAI Backend Service** — Combining OpenAI's API with Python tools to extract structured data from unstructured PDF resumes; managing API errors and context limits.
- **Video 9: Content Creation Using OpenAI Backend** — Using OpenAI's APIs to generate creative text and images, combining them into a web page for storytelling with automated publishing.
- **Video 10: Sentiment Classification Using OpenAI Backend** — Building a sentiment analysis pipeline for social media data using the Twitter API, ChatGPT and GitHub Copilot; saving tweets to CSV, analyzing with NLTK Sentiment Intensity Analyzer.

### [Discussion] Self-Study Discussion 7.1: The Evolution and Impact of Large Language Models (LLMs)

**Learning outcomes addressed:** Identify key types and uses of AI; apply AI tools to solve organizational problems; evaluate AI's role in management functions.

**Discussion statement:** As a leader/consultant in a Tier 1 city IT firm in India evaluating the adoption of LLMs (GPT-3, BERT, LLaMA) for your organization — with a diverse team of intermediate technical proficiency facing both excitement and skepticism — reflect on:
1. How would you communicate the value and potential risks of integrating LLMs to various stakeholders (tech, management, clients)?
2. What strategic or operational challenges might arise from choosing between open-source and closed-source models, or balancing bigger models with more efficient/smaller ones?
3. Share concrete examples of how foundational AI concepts (tokenization, context windows, training methods) may impact practical business outcomes.

**Suggested time:** 60 minutes. _Self-study activity — does not count toward programme completion._
